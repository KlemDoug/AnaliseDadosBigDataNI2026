## 📘 Módulo 1 (UC1): Algoritmos, Versionamento e Consulta Estruturada

Nesta primeira unidade, cobrimos a base da programação, versionamento de código e a manipulação inicial de dados:

* **Aula 01:** Introdução a algoritmos, variáveis e linguagem Python.
* **Aula 02:** Estruturas de decisão (`if`, `elif`, `else`).
* **Aula 03:** Boas práticas de versionamento com Git e GitHub (Commits, Push, Fluxos colaborativos).
* **Aula 04:** Estruturas de decisão múltiplas (`Switch / Case` e aplicações práticas).
* **Aula 05:** Estruturas de Repetição (`For`, `While`).
* **Aula 06:** Estruturas de dados (Listas, Tuplas, Sets e Dicionários).
* **Aula 07 e 08:** Criação de Funções e passagem de parâmetros.
* **Aula 09:** Tratamento de exceções com blocos `TRY` e `EXCEPT`.
* **Aula 10:** Introdução à Biblioteca Pandas para manipulação de dados em `.csv`.
* **Aula 11:** Aula dedicada à aplicação final do Projeto do módulo.
* **Projeto:** Dedicado ao estudo de caso Tanoshimi - Caso do Restaurante Japonês.

---

## 📊 Módulo 2 (UC2): Matemática Estatística, Big Data e DataViz

Nesta segunda unidade, trabalhamos a estatística descritiva, processamento de dados em larga escala e visualização (DataViz):

* **Aula 01 a 03:** Aprofundamento em Pandas (Séries e DataFrames), medidas de posição (Quartis e Mediana) e Data Query Language (DQL) via MySQL integrado ao Python.
* **Aula 04 a 07:** Modelagem de dados e junções (JOINs), arrays com biblioteca NumPy, identificação de Outliers, e uso de Boxplot/Amplitude.
* **Aula 08 a 10:** Medidas de dispersão (Variância, Desvio Padrão, Coeficiente de Variação), Assimetria, Curtose e Resolução de Problemas.
* **Aula 11 a 14:** Fundamentos de Big Data, comparação entre Pandas e Polars, otimização com arquivos `.parquet`, execução *Lazy* e uso de memória *Cache*.
* **Aula 15 e 16:** Gráficos de dispersão, Correlação de Pearson, e Fundamentos de Análise Preditiva (Regressão Linear Simples - ponte para Machine Learning).
* **Aula 17 a 19:** Conexão, modelagem e DataViz com Microsoft Power BI, introdução a DAX, melhores práticas de visualização e projeto de encerramento.
